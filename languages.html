NightZan - Languages


NightZan or more known as NightZan999 is a: student, gamer, and developer
This page will show you how many programming languages he knows and what skill sets I have :D!





HTML

I learnt HTML as my first language. I learnt the basics in about 3 months so yeah. Pretty basic language and I could create okay webpages in like 3 months!

I have coded a ton of HTML on github so yeah. Be sure to check my github out for plenty of HTML code!





CSS

Yup, I needed to style my good webpages! So I needed CSS

In about 6 months I had learnt a ton of CSS and become very advanced (not very but for 6 months). I learned hover elements and pseudoelements.

And I learned CSS variables which is kinda cool.

I kept learning this for one more year and then I had pretty much mastered it!

JavaScript

Its kinda surprising how I didn't go to something like python.

My uncle was not there to actually guide me before I picked my languages He lived abroad and I had to write emails.

And I kinda wanted to experience my coding journey myself. So I picked the choice to learn JavaScript alongside complex CSS and styling.

This was probably the dumbest decision I ever made! I did learn about loops and more complex variables and after learning let and const. I kinda gave up on JavaScript and in the end never became too good at it.

Just okay, but this much learning was somehow enough for the future (I didn't know it yet :D)





















PHP

Okay now you'd get really mad at me! I went to JavaScript and now had the guts to survive PHP.

I learned it with my friend Yash Bhatnagar (check him out here).
He is awesome at JavaScript. He wasn't so keen to learn PHP in the beginning. But then when he realized it was cool. He became better than me.

Turns out that when I learned it in detail (months later) I realized I loved writing authentic PHP which made me install PHPStorm!

Now I was an experienced programmer and knew quite some languages. (I did not have CSS to bother me now!)
So then I just continued my journey! I became an awesome PHP dev after figuring out authentication with MySQL and writing SQL queries!

Then I finally made an XAMP and WAMP server which used MySQL, Apache and PHP!



Python

Finally I learned Python. I felt really comfortable writing python. And went ahead and started using django and flask.

I still need to explore python's machine learning and AI capabilities! But now I made a discord bot! Check projects for more information

So yeah, once I finished learning this I clearly had to go further. I used SQLite3 as a database and had some fun with it!

I realized that python is a vast language with beginner friendly syntax and thus overall comfortable to write and read code in!

I started to use PyCharm for this now!

















NodeJS

So I talked about my friend Yash Bhatnagar he figured out backend gizmo with NodeJS and ExpressJS. I soon realized rather than moving to game engines.

To get good at websites is the main aim at frontend. I already new PHP so this wasn't a problem. But having a new perspective to web developement was all I needed!

The LAMP stack consists of Linux, Apache, MySQL and PHP! But the MEAN stack consists of MongoDB, ExpressJS, AngularJS and NodeJS.

I soon realized that NodeJS was also really powerful! And I quickly realized that I had to adapt to this new technology! I read up ExpressJS docs and figured it out. I also compared it to PHP and thus got a better understanding of which one I like more!

Java

SCSS
