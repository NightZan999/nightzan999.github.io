Django | Python

I used Django to build a sample site I think django is good, but not nearly as good as some other web developement frameworks. This is because most of it is actually just plain python object oriented programming!

The problem with this is that you would need a lot more modules just to write some authentication / server side.
Your interactions with the background server is only by accessing the server. This not only makes it hard to write SQL queries. But hard in general!

The models are easy to setup and its good, just that there are much better alternatives like ExpressJS.













ExpressJS | JavaScript

I learned ExpressJS as part of the MEAN stack! Yash Bhatnagar my friend has created plenty of projects with ExpressJS and the MEAN stack. Incase you don't know what the MEAN stack is. Now this got cool, NodeJS had a faster but less efficient backend than PHP. Ultimately I chose PHP but I had fun writing ExpressJS code. This is one of the reasons that got me back to JavaScript. NodeJS was another!
